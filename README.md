# Still Relevant Apparel — Website

This is a simple, fast **static website** for Still Relevant Apparel.

## Deploy to Vercel
1. Upload these files into your GitHub repo (`still-relevant-apparel`).
2. In Vercel: **Add New → Project → Import** the repo.
3. Framework preset: **Other** (static).
4. Build Command: *(leave blank)*
5. Output Directory: *(leave blank)*
6. Click **Deploy**.

Vercel will serve `index.html` automatically.
